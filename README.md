# PrepPal: AI-Driven Mock Interview Platform with Multimodal Feedback Analysis
PrepPal is an interactive platform that uses artificial intelligence to help job seekers prepare for interviews. By simulating real interview scenarios, providing personalized feedback, and analyzing your verbal and non-verbal communication, our tool helps you build confidence and improve your performance.
